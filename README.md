# 叮咚猫 · AI活力审计系统

## 这是什么？
一套轻量级的企业AI使用情况审计工具。它能自动扫描企业内部AI服务的调用日志，发现“僵尸AI”（长期未使用）、“边缘AI”（未被重用）、“卡顿AI”（性能问题）和“核心AI”（高价值资产）。

## 核心功能
- 自动感知：模拟“叮咚猫”游走，分析日志
- 活力分类：僵尸、边缘、卡顿、核心
- 报告生成：可视化展示，可导出

## 快速开始（本地运行）
1. 确保有 Python 3.8+ 环境
2. 下载本仓库代码
3. 安装依赖：`pip install flask pandas`
4. 运行演示：`python demo.py`
5. 打开浏览器访问 `http://localhost:5000`
